Q&A API Service

Добро пожаловать! Этот сервис предоставляет REST API для работы с вопросами и ответами.

Эндпоинты:

  • GET /api/questions/ — список всех вопросов
  • POST /api/questions/ — создать новый вопрос
  • GET /api/questions/{id}/ — получить вопрос и все ответы
  • DELETE /api/questions/{id}/ — удалить вопрос (с ответами)
  • POST /api/questions/{id}/answers/ — добавить ответ к вопросу
  • GET /api/answers/{id}/ — получить ответ
  • DELETE /api/answers/{id}/ — удалить ответ

Все данные возвращаются в формате JSON. Для тестирования можно использовать curl или Postman.
